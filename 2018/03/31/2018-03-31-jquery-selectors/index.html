jquery对象和dom对象

jquery对象的实质

jquery对象其实是一个javascript的数组，这个数组对象包含125个方法和4个属性

4个属性分别是

  • jquery 当前的jquery框架版本号

  • length 指示该数组对象的元素个数

  • context 一般情况下都是指向HtmlDocument对象

  • selector 传递进来的选择器内容 如：#yourId或.yourClass等
